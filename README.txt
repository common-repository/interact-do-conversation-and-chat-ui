This plugin helps you to easily embed Interact.do chat into your WordPress site.

== Description ==

Setup your interaction journey on your WordPress site with ease.  You can easily deploy your Interact.do chat web experience
without the need to create code yourself using this plugin.

[Interact.do](https://interact.do/) provides a platform to create conversational experiences with no code.  Click, drag and drop
to easily create an experience for your users that is simple and meets your needs and those of your users.

If you are new to Interact.do, learn more about the features and functionality [here](https://interact.do/features).

If you already have Interact.do, just get the appID when you are looking to deploy your journey.  You can access the appID for the journey you are
currently editing [here](https://interact.do/interaction/deploy/ "Deploy your interaction journey")

We'll even help pull it in for you if you are already logged in.


== Installation ==

= Manual Installation =

1. Upload `interact-do-ui.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('interact_do_ui_hook'); ?>` in your templates
